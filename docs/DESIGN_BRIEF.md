# AI Consulting Agency Landing Page

## Design System
- **Dark Background**: Charcoal/black (oklch 0.08)
- **Primary Accent**: Neon Green/Teal (oklch 0.68 0.22 145)
- **Secondary Accent**: Soft Cyan (oklch 0.65 0.18 190)
- **Text**: Light gray/white (oklch 0.95)
- **Cards**: Glass morphism with soft blur

## Key Sections
1. Hero with animated workflow diagram
2. Trust signal strip (auto-scroll)
3. Problem/Solution section
4. Core offerings (3 cards)
5. How it works (node-based)
6. Use cases grid
7. Trust & credibility section
8. CTA section
9. Footer

## Animation Principles
- Smooth scroll-driven motion
- Hover effects < 300ms
- No flashy animations, engineering-focused
- Node diagram with signal flow visualization
